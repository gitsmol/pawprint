What does this page do?

Pawprint uses data from Bearable to (crudely) draw graphs. It can only use data provided by you. If you want, you can make a pdf or print your graph to store or share.

What about my data?

Of course, you are very careful about sharing your Bearable data. This page does not save your data. It is set up to convert your Bearable data to a graph in memory. When you send your data to our server, it is not saved to disk. Rather, it is loaded into memory, taken apart and rearranged to present a graph. When you see the graph, the server has already 'forgotten' your data.

Having said that, you should only send in your data if you feel comfortable about it. Pawprint does not (currently) actively obfuscate data. Theoretically, if someone captures the servers memory somehow, at just the right time, your data is there. Maybe.

If you want, you can check out the very simple code behind Bearable by visiting my Github page.
